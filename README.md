\# 8-Puzzle BFS Solver



Консольна програма на C#, яка розв’язує задачу \*\*8-Puzzle\*\* за допомогою алгоритму пошуку в ширину (BFS — Breadth-First Search).



\## 📌 Опис



Програма:



\- Приймає початковий стан поля 3×3

\- Використовує алгоритм BFS для знаходження найкоротшого шляху до цільового стану

\- Виводить повну послідовність ходів

\- Збирає та відображає статистику пошуку
